# pyspark-eda-cancer-data
Student project performing Exploratory Data Analysis (EDA) on a cancer dataset using PySpark on Databricks. Focused on data transformation and quality assessment of synthetic records.

## Project files
This project was developed in Databricks.

- **`pyspark-cancer-data-EDA.ipynb`** — the notebook source (code + narrative) used to run the analysis.
  You can open it locally in Jupyter / VS Code or import it into Databricks to rerun/modify the workflow.

- **`index.html`** — an exported, static version of the notebook for easy viewing (tables/charts) without running Databricks.
  Use the link below to open the preview in a browser.
  
  [Interactive Preview](https://skulskimilosz.github.io/pyspark-eda-cancer-data/)
